FTP上传文件须知：

1、FTP登录后，网站根目录为WEB目录，请直接把网站程序放在WEB目录，不要放在其他目录。缺省网站的文档按优先顺序为 index.html,index.php。上传文件名、目录名最好是字母、数字或者是字母数字混合，请不要用中文或者怪字符命名。

2、景安虚拟主机空间包含WEB、LOG、Backup三个目录，三个目录文件存储内容如下：

WEB目录：放置您的站点文件；

LOG目录：通过“日志下载”功能将您网站的访问日志文件下载保存到此目录；

Backup目录：放置您的备份文件；可根据您的需要自行清理备份文件，以便节省使用空间。

以上目录文件不可随意删除或修改目录名，否则将导致您的网站无法正常访问！

本网页文件名为 index.html 是您的主机自动开设后系统生成的页面，您可以删除掉或用自己的页面将此页面覆盖。推荐您使用FlashfXP等软件进行上传（FlashFXP软件下载）。

3、FTP软件如何设置

FTP软件有很多种，但每种软件都要设置以下几项内容：

1〕 FTP主机地址：请输入主机管理页面中显示的FTP链接地址；

2〕 FTP 用户名：webmaster@+您开通时填写的域名或业务标志，例如webmaster@test.com；

3〕 FTP 密码：即您开通时填写的密码；

4〕 FTP 端口：默认21即可。
